# ESG Evidence Pack (Mock)

This folder demonstrates a mock ESG evidence pack structure used to support consistent responses to ESG and sustainability questionnaires.

## What this shows
- Centralised ESG evidence storage
- Clear ownership and review cadence
- Traceability between KPIs and supporting documentation

## How it fits together
- KPIs live in the ESG KPI Tracker
- Each KPI references evidence stored here
- The Evidence Index provides a single source of truth

All files are illustrative and for demonstration purposes only.
